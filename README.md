# 🎓 College Management System (Java)

## 📌 Project Description

The College Management System is a Java-based application designed to automate and simplify college administrative tasks. It manages student records, course details, and various operations efficiently using object-oriented programming and data structures.

---

## ⚙️ Features

* Add new student details
* Display all student records
* Manage course information
* Add and delete courses
* Check student attendance
* Identify students with low attendance (<75%)
* Efficient data storage using linked list structure

---

## 🛠️ Technologies Used

* Java Programming Language
* Object-Oriented Programming (OOP)
* Data Structures (Linked List)
* Scanner class for user input

---

## 📂 Project Structure

* `CollegeManagementSystem.java` – Main driver class
* `CollegeOperations.java` – Contains core operations
* `StudentNode.java` – Stores student details
* `CourseNode.java` – Stores course details

---

## ▶️ How to Run

1. Open project in any Java IDE or VS Code
2. Compile all files:

   ```
   javac *.java
   ```
3. Run the program:

   ```
   java CollegeManagementSystem
   ```
